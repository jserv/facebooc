What is facebooc?

A proof-of-concept Facebook clone written in C. The only dependency is SQLite3.

Prerequisites

Install SQLite development package:

sudo apt-get install libsqlite3-dev
#EL
sudo dnf install sqlite-devel sqlite

Build and Run

cd facebooc
make run
